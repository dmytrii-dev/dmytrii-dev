Website analytics

This website uses Vercel Web Analytics to understand how the site is used and improve its content. It measures page views and approximate daily visitors and may provide aggregated breakdowns such as the visited page, referrer, country, device type, operating system, and browser. Vercel states that Web Analytics does not use cookies, stores anonymized aggregated data, and does not create a permanent visitor identity. The data is not used for advertising, cross-site tracking, or building individual visitor profiles.
