Shipping automation

DHL Shipping Autofill

A browser automation tool that converts a raw receiver address block into structured DHL shipping form fields.

Project overview

This tool was built for DHL shipping workflows where receiver address data has to be copied manually into multiple form fields.

Instead of entering each field one by one, the user pastes a full address block and the extension fills the matching DHL fields automatically.

Problem

  • Repeated manual entry of receiver address data
  • Higher risk of typing mistakes in shipping forms
  • Slow preparation of shipments when the same workflow is repeated often

Solution

  • Parse receiver name, company, street, house number, postal code, city, and country
  • Preview the structured data before filling
  • Fill the DHL address form automatically from the browser extension

Demo note

The original workflow was built for DHL business shipping. The demo uses DHL Online Franking so the full address-to-form flow can be shown on a real DHL page without exposing client data.
