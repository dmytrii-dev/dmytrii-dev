Shipping automation

DHL Shipping Autofill

Paste a receiver address block and fill DHL shipping fields automatically.

From raw address block to filled DHL form

The extension removes repeated manual entry by parsing receiver data and applying it directly to the correct shipping fields.

1. Paste address

A full receiver address block can be pasted into the extension instead of copying fields one by one.

2. Parse fields

The tool separates name, company, street, number, postal code, city, and country into structured data.

3. Fill DHL form

The browser extension applies the parsed data to the matching DHL fields automatically.

Project context

Built for DHL business shipping workflows where receiver details were repeatedly copied into shipping forms. The demo uses DHL Online Franking so the full address-to-form flow can be shown on a real DHL page without exposing client data.
